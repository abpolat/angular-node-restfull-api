# Restful API
> Angular 2 + Node.js + MongoDB + Bootstrap

A sample project written by me to try CRUD operations with Angular 2

## Setup
    git clone https://github.com/abpolat/angular-node-restfull-api.git
    cd angular-node-restfull-api

    npm install
    npm start

## Open

    http://localhost:3000
